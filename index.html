What is npm?

  • Package manager for node.js
  • Included with node.js
  • Packages can be installed globally by passing the --global flag and installs packages in your configured path for global node module 
    • npm install {project} –-global
  • Packages can be installed locally and creates a node_modules 
    • npm install {project}
  • https://docs.npmjs.com/

package.json

- A shopping list for the kitchen supply store

package.json continued...

What is package.json?

  • A list of node.js modules to be installed
  • Thousands of node.js modules for all kinds of things like, Sass, CSS and JS linting, CSS and JS optimization, Image optimization, CSS auto prefixing, and much more
  • When npm runs it will install the package as well as any dependencies the package may have
  • Creates a node_modules folder with all of the projects
  • Uses semantic versioning: https://docs.npmjs.com/misc/semver
  • Different dependency types: dependencies, devDependencies, bundleDependencies
    • Dependencies: required to run your application
    • DevDependencies: only required develop your application
  • TIP: once your packages have downloaded, add node_modules to your .gitignore. If you plan on using a package in your production environment you should copy the assets you need out to another location. We can automate this using gulp.js
  • https://docs.npmjs.com/files/package.json

What is npm shrinkwrap?

  • Creates a locked file of node projects and their dependencies for the exact versions you are using
  • Ships with node.js
  • Creates a npm-shrinkwrap.json file
  • Use –dev to include devDependencies

What is Bower?

  • A list of bower componets to be installed
  • A list of bower componets must have an endpoint
  • When bower install runs it will download the packages and place it in a bower_components folder
  • Uses semantic versioning: https://docs.npmjs.com/misc/semver
  • TIP: once your packages have downloaded, add bower_componets to your .gitignore. If you plan on using a assets in your production environment you should copy them to another location. We can automate this using gulp.js
